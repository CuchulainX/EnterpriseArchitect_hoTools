# AddInSimple

Simple AddIn to see basic EA AddIn features:
- Running Addin Skeleton
- AddIn Search
- AddIn ShapeScript
- LINQ to SQL
- Run LINQPad Query from AddIn (output as html)
- Deployment 

Details see in: https://github.com/Helmut-Ortmann/EnterpriseArchitect_hoTools/wiki/AddinSimple
